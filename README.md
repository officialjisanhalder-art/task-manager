## ‚ú® Features

- **Full Stack CRUD APIs** ‚ÄĒ Connects directly to local/remote MongoDB databases using Mongoose
- **Zero-Config Database Fallback** ‚ÄĒ No Mongo installed? The backend auto-switches to in-memory store so it never crashes!
- **Premium Glassmorphic Panel Layout** ‚ÄĒ Neon glows, blur backdrops, responsive sidebar layout
- **Color Coded Priorities** ‚ÄĒ Visually see high/medium/low priority tags
- **Task Filters** ‚ÄĒ Instantly filter between All, Active, and Completed items
- **Progress Tracking** ‚ÄĒ Live stats and fill bar updating task completion percentages
